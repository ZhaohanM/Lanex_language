What is Lanex?

Lanex is a web application that enables users to communicate with each other to exchange languages, thereby helping to improve language knowledge and allowing users to create new language partners along the way. The web application focuses on uniting people so that they can find language partners to participate and communicate with others.

What can I do in Lanex?

Firstly, the user can set the language request based on the language of interest and preferred location. In the request, the user can set the time and place to meet with other interested users, with a map functionality provided. For online activities in Lanex, they can set the location as their residence and post it in the description or title so that they can find language partners who want to chat online on different platforms. Then, the demand can be anything related to language learning: request to meet, have an online chat, join a study group, get recommendations, find a teacher, etc.

How to find and accept a request?

Users can accept language requests created by other users to participate in language-related activities and meetings. And Lanex grants users the ability to comment on language requests in order to better meet the needs of users. If there is an other user's request which interests you, you can accept it by clicking on the respective button. Once redirected to the owner of the request's profile page, click on the Send email button. This should open up your default email client where you can mention about your interest for their created request. (Remember: a search function is also provided to make it easier to find language requests that better match preferences.)
